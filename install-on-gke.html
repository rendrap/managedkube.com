Install ManagedKube from GKE’s Marketplace

Installing ManagedKube on GKE is as easy as one click. Plus, get up to $500 in GCP credits when you use our link

1

Search for and click on ‘ManagedKube’ in the GKE marketplace or follow this link

2

Click ‘Configure’ on the ManagedKube application page

3

Pick the cluster you’d like to install ManagedKube on and click ‘Deploy’

4

Wait for deployment to finish. You’ll automatically go to the application page. If you see an External IP of 10.x.x.x, you’ll need to wait a few minutes for the external load balancer to be created. Wait for the External IP to generate and click on that IP. Note the username and password generated

5

Log-in to Grafana using your username and password

6

Click Home in the upper left hand corner

7

Click on ‘ManagedKube Kubernetes Cost Attribution’ dashboard

8

You’re done! Wait for this dashboard to populate over time as the dashboard collects information on your Kubernetes cloud spend. Use this information for budget planning and to optimize your cloud efficiency. If you have other clusters you’d like to install ManagedKube on, you’ll need to repeat this process
