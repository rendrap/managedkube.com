Thanks for signing up for k8sBot on GCP's Marketplace!

To complete your registration process, install the k8sBot app on Slack by following the instructions below. k8sBot will guide you through the rest of the process.

Easy Installation Instructions

1) Click the button to install k8sBot on Slack:

2) Invite @k8sbot to your channel by saying anything to @k8sBot

3) Ask for help (@k8sBot help) and k8sBot will guide you through the rest of the sign-up process


Once you complete this, check out the k8sBot docs to learn more about what k8sBot can do.
