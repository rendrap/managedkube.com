DevOps Internship

Description: If you want to learn how to build for the users, understand how organizations around the world work with data and solve problems for data teams, apply now!

Requirements:

  • Cloud Service Providers (AWS/Google cloud), Docker+Kubernetes ecosystem is a plus.
  • Experience programming in Golang, Python

What will you do?:

  • Collaborate with our team to manage and build upon the infrastructure that supports our products.
  • Work with our Engineering Team to build tools and establish processes that can help developers deploy and release their code seamlessly.
  • Create automation tools for building, provisioning and deploying servers.
  • Help automate systems for 24x7 monitoring and failure recovery.
  • Respond to service outages/incidents and ensure system uptime requirements/SLAs are met.
  • Work with other DevOps Engineers in the team for routine audits, patching, upgrading systems, capacity planning and benchmarking.
  • Own, scale and maintain the compute and storage infrastructure for various product teams.
