Just a few more steps to finish k8sBot installation!

1) Invite @k8sbot to your channel by saying anything to @k8sBot
2) Ask for help (@k8sBot help) and k8sBot will complete the process

k8sBot will ask for your contact information and then generate a k8sBot agent yaml file for you to install on your cluster. Once you complete this, check out the k8sBot docs to learn more about what k8sBot can do.
